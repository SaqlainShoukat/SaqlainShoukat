Data Transmission Mode

  • Define

Data transmission mode refers to the method and direction in which data is transmitted between two devices in a network.

Types

Three Moded:-

  1. Simplex Mode
  2. Half Duplex Mode
  3. Full Duplex Mode

Explanation of Modes

Each mode is suited for different applications based on the requirements for data flow and communication between devices.

1. Simplex Mode

Data transmission is unidirectional. Data flows in only one direction, from the sender to the receiver, with no provision for the receiver to send data back. Example: Keyboard to computer.

2. Half Duplex Mode

Data transmission is bidirectional, but not simultaneously. Data can flow in both directions, but only one direction at a time. Example: Walkie-talkies.

3. Full Duplex Mode

Data transmission is bidirectional and simultaneous. Data can flow in both directions at the same time. Example: Telephone calls.
